◆ wait()

int wait ( )

Wait on the barrier.

The calling thread will wait on this barrier. If calling thread is the last thread that was waiting on the barrier, it unblocks all other threads that were waiting on the barrier, otherwise it blocks on the barrier.

Returns
ZERO if calling thread blocks, NON-ZERO if it was the last thread and unblocked all other threads.
